Conseils d’utilisation

Il est possible que le nom de cette police varie légèrement, selon l’emplacement où vous l’utilisez. Voici ce que vous devez rechercher.

Bureau

Dans les menus de polices des applications, cette police sera répertoriée comme suit :

{{familyCtrl.selectedVariation.preferred_family_name}} {{familyCtrl.selectedVariation.preferred_subfamily_name}}

Web

Pour utiliser cette police sur votre site web, appliquez la CSS suivante : 

font-family: {{familyCtrl.selectedVariation.family.css_font_stack.replace('"', '').replace('",', ', ')}};
font-style: italicnormal;
font-weight: {{familyCtrl.selectedVariation.font.web.weight}};
